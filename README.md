# 投资看板

一个纯静态的投资信息看板，用本地示例数据展示市场指数、自选股、资金流向、板块趋势和财经新闻。

## 当前功能

- 大盘指数概览：上证指数、深证成指、创业板指、科创50。
- 自选股：展示默认自选列表，支持按代码或名称添加、删除，并保存到浏览器本地存储。
- 资金流向：主力资金、北向资金、板块流入/流出排行、多日趋势。
- 财经新闻：展示新闻摘要、相关板块和关键数据标签。
- 交互体验：移动端优先布局、顶部分类 tab、卡片折叠、手动刷新和自动刷新开关。

## 文件结构

```text
index.html   页面结构
styles.css   样式和响应式布局
app.js       页面交互与数据渲染
data.js      本地示例数据
```

## 本地运行

这个项目不需要构建工具。可以直接打开 `index.html`，或用本地静态服务运行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 数据说明

当前数据来自 `data.js` 的本地样例，页面展示的“数据时间”取自 `DATA_UPDATE_TIME`。这些内容仅供页面演示和投资信息整理参考，不构成投资建议。
