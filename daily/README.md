# 📊 投资日报

一个简洁、现代的投资日报单页网站，自动获取每日财经数据。

## ✨ 功能特点

- 📰 **每日财经新闻** - 自动获取最新的财经要闻（5-10条）
- 💹 **今日利好板块** - 展示资金流入最多的前6个板块
- 💰 **基金推荐** - 3-5只精选基金推荐，包含详细推荐理由
- 🔄 **自动刷新** - 每10分钟自动更新数据
- 📱 **响应式设计** - 完美适配手机和电脑
- ⚡ **纯前端实现** - 无需后端服务，数据通过JSONP跨域获取

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 现代渐变样式、响应式布局
- **原生JavaScript (ES6+)** - 数据获取和渲染
- **JSONP** - 跨域获取API数据

## 🌐 数据源

所有数据均来自公开API，使用JSONP方式跨域获取：

| 数据类型 | 数据源 | API地址 |
|---------|--------|---------|
| 财经新闻 | 东方财富网 | `newsapi.eastmoney.com` |
| 板块资金 | 东方财富网 | `push2.eastmoney.com` |
| 基金数据 | 天天基金网 | `fundgz.1234567.com.cn` |

## 🚀 使用方法

1. **直接打开** - 双击 `index.html` 即可在浏览器中运行
2. **本地服务器** - 使用任意静态文件服务器（推荐）
   ```bash
   # 使用Python
   python -m http.server 8080
   
   # 使用Node.js
   npx serve .
   
   # 使用PHP
   php -S localhost:8080
   ```
3. **GitHub Pages** - 上传到GitHub仓库，启用Pages功能即可
